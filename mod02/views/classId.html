Estudo de Class e Id

Antes de começar deixarei logo destacado que na utilização de CSS, é possível simplificar a sintaxe do código simplesmente colocando hashtag(#) antes do nome do id criado no HTML e ponto (.) antes do nome da class criada no HTML.

Além disso, a utilização dessas duas tags no HTML faz com que as CSS possuam também dois tipos de seletores, os genéricos e os personalizados.
Os genéricos são todos aqueles que não possuem nenhum atributo passado dentro de sua tag.
Em contrapartida os seletores personalizados possuem sua própria marcação (nome da Id ou o nome de uma Class) passada dentro da tag que se queira ser personalizada.
