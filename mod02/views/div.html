Organização do Site

O modelo de caixas serve principalmente para organização da página. Esse modelo permite colocar um elemento dentro do outro, pois cada elemento mostrado na tela é representado dentro de uma caixa, e assim formar uma hierarquia de tags.

Toda box tem seu tamanho contendo seu conteúdo, e todo conteúdo possui um limite chamado de border (aqui é usado border-radius: para arredondar a borda do box). Entre o conteúdo e a border existe o padding para haver uma distância entre conteúdo e border.

Box-level

Uma caixa do tipo box-level é que ele ocupa toda a largura da tela (toda a viewport) e automaticamente é iniciada em outra linha

Exemplos de Box-level:

  • div
  • main
  • nav
  • form
  • video

Inline-level

Uma caixa do tipo Inline-level é uma box menor, que ocupa a mesma linha do conteúdo anterior a ele e os próximos conteúdos após essa box Inline-level serão mostrados na mesma linha, junto com essa box Inline-level.

Esse tipo de box não ocupa toda a viewport da tela, somente a área necessária que ele necessita. Por isso é possível exibir conteúdo antes e depois dessa box.

Exemplos de Inline-level:

  • span
  • em
  • input
  • button
  • label
  • select
