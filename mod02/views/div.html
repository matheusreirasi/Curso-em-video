Organização do Site

O modelo de caixas serve principalmente para organização da página. Esse modelo permite colocar um elemento dentro do outro, pois cada elemento mostrado na tela é representado dentro de uma caixa, e assim formar uma hierarquia de tags.

Toda box tem seu tamanho contendo seu conteúdo, e todo conteúdo possui um limite chamado de border (aqui é usado border-radius: para arredondar a borda do box). Entre o conteúdo e a border existe o padding para haver uma distância entre conteúdo e border.

Box-level

Uma caixa do tipo box-level é que ele ocupa toda a largura da tela (toda a viewport) e automaticamente é iniciada em outra linha

Exemplos de Box-level:

  • div
  • main
  • nav
  • form
  • video

Inline-level

Uma caixa do tipo Inline-level é uma box menor, que ocupa a mesma linha do conteúdo anterior a ele e os próximos conteúdos após essa box Inline-level serão mostrados na mesma linha, junto com essa box Inline-level.

Esse tipo de box não ocupa toda a viewport da tela, somente a área necessária que ele necessita. Por isso é possível exibir conteúdo antes e depois dessa box.

Exemplos de Inline-level:

  • span
  • em
  • input
  • button
  • label
  • select

Border

Como estamos falando de modelo de caixas, nada mais importante que definir seu tamanho. E é aí que entra a edição das bordas das caixas.

Para deixar a borda com diferentes tipos, utilizamos o <border-style> e existem algumas que mais são usadas para o parâmetro como:

  • solid: todas as bordas (esquerda, direita, superior e inferior) possuem a mesma medida e nenhum design especial.
  • dotted: as bordas da caixa são em formato de pequenas bolas.
  • inset: as bordas possuem um aspecto de moldura, onde as bordas esquerda e superior possuem uma cor e as bordas direita e inferior possuem outra cor.
  • doble: as bordas possuem duas linhas, uma interna e outra externa.

Também é possível mesclar diferentes tipos de borda.
Para compreender melhor, acesse o site da Mozilla para entender e fazer os testes com diferentes tipos de bordas.

Para a borda realmente aparecer é preciso colocar a cor da borda, sua espessura usando border-width e seu style. Sem esses três a borda não aparece na tela.

O método mais eficiente de criar uma borda é utilizar o dentro da tag border os valores correspondentes de border-width, border-style e border-color.

A shorthand usada em border também pode ser usada em outline, seguindo o mesmo padrão.

Padding é a distância entre o conteúdo e a borda.

É bom deixar uma pequeno padding caso seja necessário colocar alguma borda em volta do conteúdo, deixar sem qualquer edição de padding faz com que o conteúdo fica muito próximo da border e isso não é visualmente agradável.

Margin

Resumidamente falando, margin é a distância entre o padding de uma box até o padding de outra box. É o tamanho final da box.
A utilização de margem faz com que toda a box se movimente, o que não ocorre com padding e border, pois são alterados os estilos somente.
