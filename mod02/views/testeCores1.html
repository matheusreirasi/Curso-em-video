Teste de cores

No site principal do estudo de CSS o degradê funcionou sem problemas, mas aqui tive que fazer algumas modificações e mesmo assim não está tão certo quanto lá.

Tags CSS desse site

Tive que usar <background-attachment: fixed> para fazer o degradê ficar colado no fundo do site. Também foi usado como configuração global a altura de 100%, talvez nem fosse preciso já que eu coloquei o degradê colado no fundo.

<background-image: linear-gradient()> posso usar quantas cores eu quiser para criar o efeito monocromático. Quando for especificar a direção também é possível indicar utilizando posições como para direita, para esquerda, para cima ou para baixo, tudo em inglês. Quando usar medidas em graus, os valores vão até noventa e se eu quiser utilizar o degradê para ir pra uma posição oposta eu uso valores negativos. Por exemplo, para o degradê ir para a esquerda eu uso <-90deg> e para ir para a direita uso <90deg>.

Também é possível dizer qual será a cor que será mais vista em tela passando sua porcentagem logo depois do código da cor. Se nenhum valor for passado em todas as cores terão o mesmo tamanho.
