Teste de cores

No site principal do estudo de CSS o degradê funcionou sem problemas, mas aqui tive que fazer algumas modificações e mesmo assim não está tão certo quanto lá.

Antes esse site estava com o css dentro do HTML e agora está em um arquivo separado. Com o CSS dentro do HTML estava dando erro e cada conteúdo dentro da tag era mostrado em uma página do site, para eu ver o resto do conteúdo eu tinha que descer a página. Agora todo o conteúdo aparece certinho, um embaixo do outro sem eu precisar descer. Esse problema apareceu porque eu defini altura para 100% e cada tag só era mostrada depois de eu rolar pra baixo de acordo com o tamanho da tela.

Tags CSS desse site

Tive que usar <background-attachment: fixed> para fazer o degradê ficar colado no fundo do site. Também foi usado como configuração global a altura de 100%, talvez nem fosse preciso já que eu coloquei o degradê colado no fundo.

<background-image: linear-gradient()> posso usar quantas cores eu quiser para criar o efeito monocromático. Quando for especificar a direção também é possível indicar utilizando posições como para direita, para esquerda, para cima ou para baixo, tudo em inglês. Quando usar medidas em graus, os valores vão até noventa e se eu quiser utilizar o degradê para ir pra uma posição oposta eu uso valores negativos. Por exemplo, para o degradê ir para a esquerda eu uso <-90deg> e para ir para a direita uso <90deg>.

Também é possível dizer qual será a cor que será mais vista em tela passando sua porcentagem logo depois do código da cor. Se nenhum valor for passado em todas as cores terão o mesmo tamanho.


Na tag main eu usei:

  • <border-radius: 10px> que faz com que as pontas dessa tag fiquem arredondadas, acho que normalmente se usa 10px.
  • <padding: 20px; da um espaçamento entre o fim da tag e a última palavra da linha
  • <margin: auto;> no vídeo do Gustavo o conteúdo da página se adaptava de acordo com seu tamanho, no meu código somente deixou a tag main no centro da página.
  • <box-shadow: 10px 20px 20px 001c55aa;> a sintaxe é: primeiro coloca o quanto a sombra se estende para a direita, depois o quanto ela se estende para baixo, depois o quanto ela se aprofunda e por último a cor. Eu alterei o código para deixar o azul mais transparente e não deixar uma sombra tão aparente, somente um detalhe.

Dentro das tags <li> e <p> eu usei <text-align: justify;> pois faz parecer mais confortável para ler.
