A melhor config para a imagem de fundo é colocar como background-size cover e position center center, pois em qualquer tamanho de tela, a imagem irá se adaptar para se encaixar em toda a tela sem ter que ficar cortada de forma errada, no caso desse site, aparecer o fundo preto.
Dependendo do site que for feito, pode ser mais vantajoso colocar como background-size a opção auto ou contain.



No caso de haver a necessidade de rolar a página, usar background-attachment com a opção fixes vai permite que eu leia o restante do conteúdo sem repetir a imagem.
Para ver esse efeito funcionar é só diminuir a tela.


A shorthand para o background é : color image position repeat size attachment

Na documentação oficial diz para usar size depois de repeat na shorthand, mas atualmente não está funcionando desse jeito. Para funcionar tem que colocar background-size fora da shorthand.
