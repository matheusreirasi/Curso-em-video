Áudio no site

Quando utilizar áudio no site, deve-se colocar do mesmo jeito que coloca uma imagem dentro do site.

Existem diversos atributos que podem ser passados dentro da tag <audio> como: autoplay, controls, loop , preload.

Em alguns sites esses atributos podem não funcionar direito, no meu caso autoplay não está funcionando.

O atributo preload possui 3 valores:

  • none: indica que o usuário não necessitará consultar o áudio ou que o servidor quer minimizar seu tráfego; em outros termos indica que o áudio não deve ser pré-carregado
  • metadata: indica que embora o usuário não necessitará consultar o áudio, pegar os meta-dados (ex: comprimento) é interessante
  • auto: indica que o usuário necessita ter prioridade; em outros termos isso significa que, se necessário, o áudio inteiro pode ser baixado, mesmo que não seja esperado a execução

Quando não é passada a tag <preload> , o código é automaticamente considerado como auto, o que deve ser evitado.

Vídeos no site

Assim como em imagens, é possível utilizar hospedagem interna e externa. Quando se trata de vídeos, é sempre melhor utilizar hospedagem externa para deixar o site mais leve. Se não for possível ou não preferível utilizar hospedagem externa, deve-se optar pela qualidade de imagem de até 720p60 pois arquivos muito grandes geram despesas maiores para a hospedagem.

Arquivos M4V são os mais leves, depois vem o WEBM e o mais pesado é o MKV

Como em arquivo de áudio, sempre deve colocar mais de uma extensão de arquivo para ser carregado no site.

Eu coloquei uma foto de capa antes de dar play no vídeo usando o atributo poster, porém ela não tem o mesmo tamanho do vídeo.

Vídeos externos não precisam ficar dentro da tag <video>.

Quando for trocar de vídeo no site, pode ocorrer de mesmo quando colocar o novo código do site, aparecer o vídeo antigo. Quando isso acontecer só precisa fechar o arquivo HTML e abrir de novo.

Outra opção de incorporação de vídeo externos são Vimeo e VK. O Vimeo possui mais opção de personalização e a qualidade de imagem do vídeo também são melhores que a do Youtube, porém, por ter um codec diferente, pode ocorrer de o vídeo dar umas pequenas travadas durante a reprodução.

CSS

Aqui começa a beleza do site

Antes de começar vale destacar as tags que não são mais usadas em CSS:

Existem 3 maneiras de usar CSS: inline, interno e externo

  • Inline:é alterado a estilo do site somente naquela linha, ou seja, na tag correspondente. Só precisa escrever "style" dentro da tag que será alterada e colocar todos os parâmetros dentro de uma string. Para separar uma tipo de edição de outro basta colocar ponto e virgula entre eles. <style="text-align:right; background-color: blue;"
  • Interno: a alteração do estilo é feito dentro do <head>. Para isso basta colocar a tag style, uma abrindo a tag e outra fechando. Dentro do style será colocado a tag do HTML que será editado e depois dessa tag coloca chaves para alterar toda a tag do HTML dentro das chaves.
  • Externo: a forma que é feito é igual ao interno, porém é feito em outro arquivo.

Quando for editar algo com CSS, as tags inline se sobreporão sobre as outras tags interna e externas. Se não houver uma edição inline, a edição interna se sobreporá em cima da edição externa.

Quando for colocar as regras no topo da CSS tem que colocar com aspas duplas.

Quando eu coloquei o CSS em uma pasta diferente da pasta do HTML que será editado o arquivo não foi carregado na página
